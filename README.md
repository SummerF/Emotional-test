# airanTest

#### 项目介绍
airan测试项目

#### 软件架构
软件架构说明
Vue.js 2.5 	基础语法，组件
Vue-cli 3.0 Vue 官方脚手架
element-ui	某些地方的ui布局
axios 		后台数据获取 + 本地模拟后台数据获取
better-scroll 项目整体滑动

#### 安装教程

1. git clone git@gitee.com:FengOrigin/airanTest.git
2. 需安装官方脚手架 Vue-cli 3.0以上

#### 使用说明

1. cd project 
2. npm run serve
3. npm run build
